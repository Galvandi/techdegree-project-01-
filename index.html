Background

I'm a 26 year old guy that is determined to become a Web Developer by the end of 2019.

Learning to code

I've discovered programming as a hobby in 2015. It was a frustrating process, but also extremely satisfying to see ideas take shape.

First I learned Blueprint Visual Scripting, which is a programming method that is used by the game engine UE4 It was a great way to get familiar with programming concepts. Soon after that I learned the basics of C++ and was able to create some quite complex game mechanics inside UE4

But Outside UE4, I never got beyond C++ console applications. I had the desire to create something more than that...

Diving into Web Development

JavaScript is a quirky language that looked familiar, didn't require compilation and ran directly in the browser. I loved it!

Functional programming, hoisting, execution contexts, closures, prototypal inheritance, even though Javascript looked familiar there were, and still are, alot of interesting new things to learn.

As for HTML and CSS... I struggle with it the most. And that, is exactly the reason why I choose Treehouse's Front-End Web Developer Techdegree.
